# Prompt

Use the filesystem tool immediately. Create a self-contained `index.html` reaction-time game with no external assets, packages, or network calls.

The page must have a `Start` button, wait for a random interval, turn the game panel green, and show a measured result in milliseconds with a `Play Again` button when the player clicks the green panel. Then verify that `index.html` exists.

This task tests an observable browser interaction rather than only static HTML output. The committed artifact is the output from a real MA + Agora run; the repository's verifier then drives the resulting page in Chromium.
