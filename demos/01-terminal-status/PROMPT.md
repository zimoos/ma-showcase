# Prompt

Use the filesystem tool immediately to create one self-contained terminal-style status page. It must visibly show an Agora provider, an available model, a few tool capabilities, and a verified completion state. Then verify that the generated `index.html` exists.

The purpose of this task is deliberately small: the resulting artifact makes the agent's filesystem action and verification observable without hiding any work behind a mock result. Larger tasks are recorded only after this baseline remains stable.
